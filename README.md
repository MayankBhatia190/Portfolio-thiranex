# Full-Stack Personal Portfolio

A complete full-stack portfolio website with admin panel, MongoDB database, and JWT authentication.

## Features
- Responsive frontend with dynamic project display
- RESTful API with Express.js
- MongoDB database for project storage
- JWT authentication for admin panel
- Skills visualization with Chart.js
- CRUD operations for projects

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript, Chart.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Authentication**: JWT, bcryptjs
